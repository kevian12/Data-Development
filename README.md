# 女装电商销售数据ETL与分析项目

## 📋 项目概述

这是一个完整的数据开发实战项目，展示了从原始电商销售数据到业务洞察的全流程。项目模拟真实工作场景，实现了数据抽取、清洗、存储（ETL）和数据分析的完整 pipeline。

**核心价值体现：**
- **数据开发全流程实践**：涵盖从原始数据到分析就绪数据的完整处理流程
- **业务导向分析**：所有分析围绕实际业务问题展开，体现数据驱动决策思维
- **技能综合展示**：融合Python数据处理、MySQL数据库操作和SQL分析三大数据开发核心技能

## 🎯 项目目标

### 技术目标
1. 使用Python Pandas完成数据清洗与预处理
2. 设计并创建MySQL数据库表结构
3. 实现数据从CSV到数据库的完整导入流程
4. 编写复杂SQL查询进行多维度业务分析

### 业务目标
1. 分析销售趋势，识别最佳销售时期
2. 识别畅销商品和热门属性，为选品提供依据
3. 理解客户购买行为，为营销策略提供数据支持

## 📊 数据集说明

### 数据来源
本项目使用kaggle女装电商销售数据，包含2022年6月至9月的销售记录。

### 数据字段
| 字段名 | 类型 | 描述 | 示例 |
|--------|------|------|------|
| order_id | INT | 订单唯一标识符 | 1 |
| order_date | DATETIME | 订单日期和时间 | 2022/6/1 16:05:00 |
| sku | VARCHAR | 商品编号 | 708 |
| color | VARCHAR | 商品颜色 | Dark Blue |
| size | VARCHAR | 商品尺码 | XL |
| unit_price | DECIMAL | 商品单价 | 298.00 |
| quantity | INT | 购买数量 | 1 |
| revenue | DECIMAL | 销售额（单价×数量） | 298.00 |

### 数据规模
- 总记录数：527条
- 时间范围：2022年6月1日 - 2022年9月9日
- 包含商品数：约20个不同SKU
